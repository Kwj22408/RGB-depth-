采集数据时的注意事项：
1.采集数据时最好直接就采集彩色点云数据
2.若采集不了彩色点云数据，在采集的过程中需要采集rgb数据和depth数据，但值得注意的是采集过程中要将二者分辨率调为一致，但即便这样在后续合成的效果中还是会有一定的误差。

getColor是获取1920×1080的rgb数据
getDepth是获取1920×1080的depth数据
geipointCloud是获取彩色点云数据
d2ccloud：将depth数据和color数据转为彩色点云数据，其需要相机的内外参数
multi_device多相机采集depth和rgb数据
two_devices_sync:两相机的同步性 读取 JSON 配置文件，为每台相机设置独立的同步参数（主 / 从模式、帧延迟、触发模式等）；
实现多台 Orbbec 相机的硬件 / 软件同步采集（保证各相机帧的时间戳对齐）；
实时采集并显示每台相机的彩色流 + 深度流（和之前的多设备程序一致）；
打印每帧的时间戳信息（用于验证同步效果）；
支持最多 2 台相机，安全处理设备启停和资源释放。
